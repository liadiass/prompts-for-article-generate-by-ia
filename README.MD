Projeto com o objetivo de gerar um artigo técnico com um layout rico, leitura agradável e com foco em promover sua autoridade técnica.

<a href="https://web.dio.me/articles/dominando-python-um-guia-abrangente-para-iniciantes-e-alem?back=%2Farticles&open-modal=true&page=1&order=oldest" title="View PDF now"> 📕Clique aqui para ler o artigo</a>

## 💻 Tecnologias utilizadas no projeto

- [ChatGPT](https://chat.openai.com/) - para título e conteúdo
- [Lexica.art](https://lexica.art/) - para gerar imagens

## 📄 Prompts e ferramentas


ChatGPT：

|   Ação   | prompt                                                                                                                                                                                                                                                                         |
| :------: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
|  título  | Crie 10 headlines para nomes de artigos sobre python                                                                                                                                                                                                   |
| conteúdo | Escreva um artigo sobre o tema "Dominando Python: Um Guia Abrangente para Iniciantes e Além"   [REGRAS]  >Vocabulário simples voltado para a comunidade tech >Texto deve ser voltado para os iniciantes em python >Texto deve abordas todas as possibilidades que python oferece para os desenvolvedores  |


Lexica.art：

- No léxica utilizamos o acervo público de imagens geradas por outras pessoas

• Angular



## ✨ Features

- Conteúdo gerado via ChatGPT
- Imagens do acervo público geradas via Lexica.art

## 📚 Materiais

- prompts utilizados

## 🛠️ Instruções de execução

Utilize os prompts acima nas ferramentas sugeridas para gerar o material base e utilize uma ferramenta de edição de documentos como power point, libreoffice , indesign para diagramação, o passo a passo em vídeo pode ser conferido na plataforma da [DIO](https://dio.me).
